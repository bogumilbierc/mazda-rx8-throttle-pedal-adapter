# M113 Throttle pedal

## Pedal plug (pedal side)

![pedal plug picture](./m113-thottle-pedal-plug.png)

## Plug pinout

| pin number | sensor                 | function      |
|------------|------------------------|---------------|
| 1          | main                   | +5V           |
| 2          | main                   | +5V           |
| 3          | secondary              | GND           |
| 4          | secondary (up to 2.5V) | sensor output |
| 5          | main(up to 5v)         | sensor output |
| 6          | main                   | GND           |
